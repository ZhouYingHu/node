2、css换行加省略号

多行

```css
text-overflow: ellipsis;
display: -webkit-box;
-webkit-box-orient: vertical;
-webkit-line-clamp: 2;
/* 这里是超出几行省略 */
overflow: hidden;
```

单行

```css
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
```
